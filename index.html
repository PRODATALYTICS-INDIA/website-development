Some of the tools we are fond of and familiar with include:

Open source tools

Python: The world's most popular programming language for data science, machine learning, and AI, used by data scientists, researchers, and developers globally in academia, tech companies, and enterprises.

R Studio: Integrated development environment for R programming, widely used by statisticians, data analysts, and researchers in academia, healthcare, finance, and government agencies for statistical analysis and data visualization.

JASP: Free and open-source statistical software for Bayesian and frequentist analysis, popular among researchers, psychologists, and social scientists in universities and research institutions worldwide.

GitHub: World's largest code hosting platform for version control and collaboration, used by millions of developers, data scientists, and software engineers globally for managing code repositories and collaborative development.

Analytical dashboards

Power BI: Microsoft's business analytics service for creating interactive dashboards and reports, used by business analysts and executives for data-driven decision making.

Tableau: Leading data visualization and business intelligence platform for creating interactive dashboards, used by data analysts and BI professionals globally.

Dashboards with Shiny: R-based web application framework for building interactive dashboards and data visualizations, widely used by data scientists and researchers in academia and healthcare.

Dashboards with Streamlit: Python-based open-source framework for building interactive web applications and dashboards, popular among data scientists and ML engineers for quickly creating data-driven web apps.

Enterprise analytics tools

IBM SPSS: Statistical analysis software for predictive analytics and data mining, extensively used by market researchers, social scientists, and business analysts in corporations, government agencies, and academic institutions.

SAS: Enterprise analytics software suite for advanced analytics, business intelligence, and data management, trusted by Fortune 500 companies, banks, pharmaceutical companies, and government organizations worldwide.
